# Research Project Profile

Use this profile when NexuBrain is configured for a paper, experiment, literature workflow, or research project.

## Goal

Keep claims, sources, experiments, assumptions, limitations, and decisions traceable.

## Start Mode Emphasis

- Knowledge indexes for theories, concepts, methods, and topic notes.
- Evidence chains for claims and citations.
- Logs for experiments, failed attempts, and interpretation changes.
- Current snapshot for latest hypothesis, verified facts, active limitation, and next action.
- Workspace entries for paper drafts, datasets, code repositories, and experiment folders.

## Integrate Mode Default

Write hidden review and proposal files under `.nexubrain/integration/` first. Preserve existing paper folders, literature notes, citation files, datasets, code, and draft content unless the human explicitly approves a native edit.
