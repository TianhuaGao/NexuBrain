# Content Project Profile

Use this profile when NexuBrain is configured for videos, essays, social posts, courses, or public communication projects.

## Goal

Connect creative planning with evidence-aware claims, production status, publication metadata, and review boundaries.

## Start Mode Emphasis

- Knowledge records for durable concepts, audience context, claims background, and topic notes.
- Task records for scripts, storyboards, recording, editing, review, and publication.
- Evidence records for public claims.
- Workspace records for source assets, editing projects, exports, and publication links.
- Logs for production decisions, feedback, analytics, and retrospective notes.

## Integrate Mode Default

Write hidden review and proposal files under `.nexubrain/integration/` first. Preserve existing scripts, assets, drafts, exports, publication notes, and project folders unless the human explicitly approves a native edit.

## Safety Notes

- Do not expose private files, credentials, unpublished sensitive work, or personal data in public media.
- Avoid unsupported promotional claims.
