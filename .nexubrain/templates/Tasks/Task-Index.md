# Task Index

A task is an actionable, inspectable, and transferable unit of work.

## Status Convention

- inbox: collected but not clarified.
- active: currently being worked on.
- waiting: blocked by external input or dependency.
- done: completed.
- paused: intentionally paused.
- archived: no longer active, kept for traceability.
