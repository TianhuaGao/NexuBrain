# Knowledge Index

Knowledge is the long-term semantic memory layer of NexuBrain. It stores durable concepts, facts, theories, topic notes, and domain understanding.

Knowledge is not the same as tasks. A knowledge item becomes a task only when the human wants to act on it, such as organizing, verifying, writing, producing, migrating, or publishing.

## Boundaries

- Put durable concepts, theories, facts, and domain summaries here.
- Put actionable work in [[Tasks/Task-Index|Tasks]].
- Put real execution locations in [[Workspaces/Workspace-Index|Workspaces]].
- Put supporting claims and sources in [[EvidenceChain/Evidence-Chain-Index|Evidence Chain]].
- Put reusable materials, datasets, templates, links, and assets in [[Resources/Resource-Index|Resources]].
- Register external knowledge bases in [[BrainRegions/Brain-Region-Connection-Index|Brain Regions]].
