# Log Index

Logs preserve process. Snapshots preserve current state.

## Log Types

- Work log: task progress, commands, failed attempts, results.
- Development log: implementation, tests, commits, rollback notes.
- Research log: hypotheses, derivations, experiment design, literature evidence.
- Agent log: rules, skills, tools, and results used by agents.
