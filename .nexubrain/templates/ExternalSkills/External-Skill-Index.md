# External Skills

This area is used to register, evaluate, adapt, and isolate external skills.

External skills do not automatically become main-brain rules. Register, evaluate, and test them here before deciding whether they should affect formal tasks or become entries in [[InternalSkills/Internal-Skill-Index|Internal Skills]].

## Areas

- Pending Skills: evaluate candidates before use. Template source: `.nexubrain/templates/ExternalSkills/Pending-Skills/_Template-External-Skill-Evaluation.md`
- Accepted Skills
- Adaptation Notes
- Execution Logs: template source `.nexubrain/templates/ExternalSkills/Execution-Logs/_Template-Execution-Log.md`
- Deprecated Or Disabled

## Evaluation Dimensions

| Dimension | Question |
|---|---|
| Source | Where does the skill come from, who maintains it, and what version is it? |
| Permissions | Does it require file writes, network access, scripts, credentials, or privileged actions? |
| Dependencies | Does it require packages, MCP services, browsers, API keys, or external services? |
| Output | Is the output inspectable, reusable, and reversible? |
| Evidence | Does it separate facts, inferences, assumptions, and citations? |
| Style | Could it pollute main-brain naming, structure, or writing style? |
| Risk | Could it delete data, leak secrets, fabricate evidence, over-automate, or exceed authority? |
