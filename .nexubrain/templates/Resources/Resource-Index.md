# Resource Index

Resources register reusable materials, templates, reference documents, images, datasets, or download locations. Durable concepts, theories, facts, and topic notes belong in [[Knowledge/Knowledge-Index|Knowledge]].

## Rules

- Do not record passwords, tokens, verification codes, or private credentials.
- For large resources, prefer recording the location instead of copying files into the repository.
- External resources should include source and access date.
- Resources related to a task should link back to the task or workspace.
- Resources related to durable understanding should link to the relevant knowledge entry.
