# Source Type Conventions

## Primary Sources

- Source code.
- Raw experiment output.
- Official documentation.
- Original paper text.
- Issue, pull request, commit, or release.
- Facts explicitly confirmed by the human.

## Secondary Sources

- Summary articles.
- Tutorials.
- Blog posts.
- Community discussions.
- Agent-generated explanations.

## Recording Requirements

- Technical and research conclusions should prefer primary sources.
- When using secondary sources, record their limitations.
- Agent inference alone cannot be treated as Verified evidence.
- Command output may only record actual executed results; do not fill in outputs that were not observed.
