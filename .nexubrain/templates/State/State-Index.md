# State Index

The state layer stores current cognitive state, working context, durable decisions, risk awareness, and routes into knowledge, tasks, and workspaces.

## Entry Points

- [[State/Current-Snapshot|Current Snapshot]]
- [[Tasks/Task-Index|Tasks]]
- [[Workspaces/Workspace-Index|Workspaces]]
- [[Knowledge/Knowledge-Index|Knowledge]]
- [[State/Decision-Log|Decisions]]
- [[State/Risks-And-Open-Questions|Risks And Questions]]

## Usage Rules

1. Put current working facts, attention focus, active task set, and next action in [[State/Current-Snapshot|Current Snapshot]].
2. Put concrete work items in [[Tasks/Task-Index|Tasks]].
3. Put real execution environments in [[Workspaces/Workspace-Index|Workspaces]].
4. Put durable concepts, theories, facts, and topic notes in [[Knowledge/Knowledge-Index|Knowledge]].
5. Put durable decisions in [[State/Decision-Log|Decisions]].
6. Put unresolved risks and questions in [[State/Risks-And-Open-Questions|Risks And Questions]].
